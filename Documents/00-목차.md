## 📊 프로젝트 요약

| 항목 | 값 |
|-----|-----|
| 플랫폼 | iOS 15.0+, macOS 12.0+, tvOS 15.0+, watchOS 8.0+, visionOS 1.0+ |
| 언어 | Swift 6.0 |
| 빌드 시스템 | Tuist, SPM |
| 아키텍처 | Actor 기반 동시성 |
| API 유형 | 동기/비동기 모두 지원 |
| 외부 의존성 | 없음 (Core 모듈) |

## 🚀 빠른 시작

```swift
import TraceKit

// await 없이 바로 사용 (권장)
TraceKit.info("앱이 시작되었습니다")
TraceKit.error("오류 발생", category: "Network")
```
